Build a modern, responsive web application called "AI Workplace Productivity Assistant" with a clean, minimal SaaS UI. The app must feature a sidebar navigation layout and a dashboard hosting three core functional tools:

1. Meeting Notes Summarizer:
   - Provide a large text-area box for pasting raw transcripts or notes.
   - Implement a "Summarize" button with dynamic loading spinner indicators.
   - Separate the generated response into clean, styled visual cards for "Executive Summary", "Key Decisions Made", and "Action Items & Deadlines".

2. AI Task Planner / Scheduler:
   - Provide an input interface to drop a raw list of tasks and dates.
   - Organize the layout visually into distinct priority categories (High, Medium, Low Priority).
   - Display a dedicated highlight panel at the bottom showing generated "Time Optimization Tips".

3. AI Chatbot Interface:
   - Build a smooth conversational window streaming user and assistant chat bubbles.
   - Add a sticky message input bar at the bottom supporting multi-turn continuous messaging.
   - Include a row of interactive prompt chips above the input (e.g., "Draft a formal email", "Help me brainstorm").

Ensure a sleek, cohesive corporate theme using deep navy and professional blue tones. Include a small footer disclaimer: "AI-generated content may require human review."
